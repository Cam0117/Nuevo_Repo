Para poder hacer uso de los archivos .py se debe primero cargar el 
documento datos_2021.csv que es la base que se analiza, posterior a esto 
se debe asegurar de importar correctamente todas las librerias necesarias 
para poder visualizar los outputs correctos. El proyecto fue realizado 
desde Google colab por lo que para una funcionalidad óptima se recomienda 
este medio para hacer uso de los códigos.
